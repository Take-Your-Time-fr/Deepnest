# ![SVGNest](http://svgnest.com/github/logo2.png)  

**SVGNest**：一个基于浏览器的矢量排样工具。  

**Demo:** http://svgnest.com  

需要SVG和Webworker支持；移动设备警告：运行演示会占用大量CPU。
